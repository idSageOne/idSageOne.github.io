Site header

2013, web design

This header was a big experiment in web design and one of the first mock-ups of this kind. It was made for XGM Community and you can see the main menu with all the key elements of the site. Scrollable gallery was a new feature not present on the site before. The work process was fairly straight forward, but you can see a novice’s work: the elements aren’t properly margined or aligned and something is lacking. Well, everyone has to start somewhere.

Portfolio site

2017, web design

This very site that you’re browsing is the result of my efforts in learning HTML and CSS. It is very minimalistic and uses only 7 colors. The total time it took to learn and create is about 3 weeks. Also it is responsive (to a degree), which I think is pretty neat.
