The Last Apprentice

2014, pixelart and animation

One of my first projects with GameMaker: Studio, the Last Apprentice was the starting point in learning to code and create pixel sprites. The game was intended to be an isometric RPG, where you play as a sorcerer in a fantasy world. During the development process I was overwhelmed with the goals and expectations so the project was never released, however it was a stepping stone towards my other works.

Winter Castle

2015, pixelart and animation

This was intended as a small platformer for a game jam, but I ended up spending all the time on mock-ups and pixelart. Although the game was never started, I was satisfied with mockups as I discovered some new art techniques when drawing platforms and ground.

Metroidvania game

2016-now, pixelart and animation

Based on lore and various designs of the Last Apprentice, this is the "dream project" that I've been working on since 2016. The mock-ups for the 2d levels may actually not end up in the final project, because I am making a working prototype and all these designs have to be redrawn to fit the current aspect ratio.

Spaceships

2016, pixelart

I've designed several pixel spaceships as a fun side-project. The main idea was that every spaceship consisted of 4 parts and all of them are interchangeable. This gave over 100 combinations with just a couple of variations for each part. The work process was intuitive and very enjoyable. Maybe someday I’ll make a space shooter...

Space Station Cleanup

2017, pixelart and animation

A 48-hour game jam I couldn’t finish in time, but thoroughly enjoyed the process. It was a shooter-platformer with robots and explosions. Nothing big planned, but while I tried to make it pretty, time ran out. The back-end systems like leveling and random buffs were finished, but there were no enemies yet. The project sits on a shelf at the moment, but I intend to work on it in the foreseeable future.
